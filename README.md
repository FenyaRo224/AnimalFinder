Десктопное приложение на C# + WinForms + Supabase для помощи в поиске потерянных питомцев.

## Основные возможности
- Просмотр объявлений о пропавших и найденных животных
- Фильтры по виду, породе, возрасту, цвету, размеру, полу
- Создание объявлений в два шага (основные данные → описание + фото)
- Загрузка фото в облако Supabase Storage
- Отображение контактов автора (только для авторизованных пользователей)
- Поиск по тексту и фильтрация

## Технологии
- C# + .NET 8 + WinForms
- Supabase (PostgreSQL, Auth, Storage)
- Асинхронные операции (async/await)
- Простой интерфейс с карточками объявлений

1. Скачай репозиторий (Code → Download ZIP)
2. Распакуй архив

4. Запусти проект (F5)
